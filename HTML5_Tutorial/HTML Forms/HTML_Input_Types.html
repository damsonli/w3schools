Input Type Submit

This type defines a button for submitting form data to a form-handler.

The form-handler is typically a server page with a script for processing input data.

The form-handler is specified in the form's action attribute.

If you omit the submit button's value attribute, the button will get a default text.

HTML5 Input Types

HTML5 added several new input types:

  • color
  • date
  • datetime-local
  • email
  • month
  • number
  • range
  • search
  • tel
  • time
  • url
  • week

New input types that are not supported by older web browsers, will behave as type "text".

Input Type Number Restrictions

Here is a list of some common input restrictions (some are new in HTML5):

Attribute HTML5 Description
disabled Specifies that an input field should be disabled
max Yes Specifies the maximum value for an input field
maxlength Specifies the maximum number of character for an input field
min Yes Specifies the minimum value for an input field
pattern Yes Specifies a regular expression to check the input value against
readonly Specifies that an input field is read only (cannot be changed)
required Yes Specifies that an input field is required (must be filled out)
size Specifies the width (in characters) of an input field
step Yes Specifies the legal number intervals for an input field
value Specifies the default value for an input field

Input Type Range

This type defines a control for entering a number whose exact value is not important (like a slider control). Default range is 0 to 100. However, you can set restrictions on what numbers are accepted with the min, max and step attributes.
